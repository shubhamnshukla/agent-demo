# Download & Installation

Luumen is a desktop application — free to get started.

## Supported Platforms
- macOS
- Windows
- Linux x64 (AppImage)
- Linux ARM64 (AppImage)

## Linux System Requirements
- glibc 2.28 or higher
- Compatible distros: Debian 10+, Fedora 29+, Ubuntu 20.04+, OpenSUSE 15.3+, RHEL 8+

## Download Links
- Download page: https://www.luumen.ai/download
- Linux x64 AppImage: at download page (click "Download for Linux")
- Linux ARM64 AppImage: at download page under "Other builds"
- macOS & Windows: at download page — select the OS tab

## Getting Started
1. Download for your OS from https://www.luumen.ai/download
2. Install and launch the application
3. Sign in via OAuth (Google or Microsoft)
4. Create a workspace and add hosts
5. Use LuumenAI for SSH access and automation
